# Documentation Index

This directory contains all documentation for the Lego City Mod, organized by purpose.

## Directory Structure

```
docs/
├── design/          # Design documents (for developers)
│   ├── GDD.md       # Game Design Document - Complete design specification
│   └── DESIGN.md    # Technical Design Documentation - Implementation details
│
├── wiki/            # Player documentation (for users)
│   ├── PLAYER_GUIDE.md  # Complete gameplay guide and getting started
│   ├── BUILDINGS.md     # Detailed building information and reference
│   ├── RECIPES.md       # All recipes and automation tips
│   └── FAQ.md          # Frequently asked questions and troubleshooting
│
└── release/         # Release documentation (for mod portal)
    ├── MOD_DESCRIPTION.md  # Mod Portal description and marketing text
    └── CHANGELOG.md        # Version history and changes
```
